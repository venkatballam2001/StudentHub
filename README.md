# 🎓 StudentHub

A modern Full Stack Student Management System built using **React.js, Node.js, Express.js and SQLite**.

StudentHub allows users to manage student records with complete CRUD functionality through a clean and responsive user interface.

---

## 🚀 Features

- ➕ Add Students
- ✏️ Update Student Details
- 🗑 Delete Students
- 🔍 Search Students
- 🔤 Sort Students (A → Z)
- 📱 Responsive Design
- 🔗 React Router Navigation
- 🌐 REST API Integration using Axios
- 💾 SQLite Database Storage

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/venkatballam2001/StudentHub.git
```

### Frontend

```bash
cd Student-Manager-Frontend
npm install
npm start
```

### Backend

```bash
cd Student-Manager-Backend
npm install
node app.js
```
